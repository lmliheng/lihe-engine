# 🌈里河引擎🌈

## 💡介绍💡
里河引擎🚀是一个用原生JavaScript打造的炫酷导航页，它集成了多个主流搜索引擎🔍，如必应、GitHub🌟（特别优化）、谷歌等，旨在为用户提供一个快速、高效的信息检索入口。

## 🌐在线访问🌐
[🚀访问里河引擎🚀](http://hao.liheng.work) 

## 🎯优点🎯
- 🔍**多引擎搜索**: 集成多个搜索引擎，满足不同场景下的信息查询需求。
- ⚡**快速访问**: 可设置为浏览器默认首页，一键直达常用网站，提升工作效率。

## 🛠️技术栈🛠️
- **前端**: 原生JavaScript + CSS
